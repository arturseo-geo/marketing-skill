# Security Policy

## Scope

This repository contains a Claude Code skill — a set of markdown instructions that guide AI marketing output generation. It does not contain executable code, API keys, or credentials.

## Reporting a Vulnerability

If you discover a security concern (e.g., prompt injection risks, instructions that could cause Claude to leak sensitive data, or any content that could be exploited), please report it responsibly:

1. **Do not** open a public issue
2. Email the maintainer or open a private security advisory via GitHub
3. Include a clear description of the concern and steps to reproduce

## What Counts as a Security Issue

- Instructions that could cause Claude to expose user credentials, API keys, ad account IDs, or private data
- Prompt injection patterns embedded in templates or examples
- Content that could cause Claude to bypass safety guidelines
- References to external resources that have been compromised
- Templates that could leak customer PII through UTM parameters or tracking

## Response

We aim to acknowledge security reports within 48 hours and resolve confirmed issues within 7 days.
