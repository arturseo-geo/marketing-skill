# Ad Copy Templates & Best Practices

## Google Search Ads (RSA)

### Specs (2026)
- **Headlines**: Up to 15 (30 characters each, min. 3 required)
- **Descriptions**: Up to 4 (90 characters each, min. 2 required)
- Google assembles combinations dynamically; pin critical headlines to position 1 or 2

### Headline Categories (write 2-3 of each)
| Category | Example |
|---|---|
| Brand/product | "Acme Marketing Platform" |
| Keyword match | "{KeyWord:Marketing Software}" |
| Benefit | "Get 3x More Leads" |
| CTA | "Start Your Free Trial Today" |
| Social proof | "Trusted by 10,000+ Teams" |
| Differentiator | "No Contracts. Cancel Anytime." |
| Question | "Struggling With Low ROAS?" |
| Stat/number | "47% Lower CPA Guaranteed" |

### Description Best Practices
- Description 1: Expand on the core benefit + credibility
- Description 2: Secondary benefit + CTA
- Include the target keyword naturally
- End with a CTA verb: "Start," "Get," "Try," "Book," "Download"

### RSA Template
```
Headlines:
H1 (pinned): [Brand/Product Name]
H2 (pinned): [Primary Benefit — 30 chars]
H3: [CTA — "Start Free Trial Today"]
H4: [Social Proof — "Trusted by 10K+ Teams"]
H5: [Differentiator — "No Contracts"]
H6: [{KeyWord:Default Headline}]
H7-H15: [Mix of benefit, question, stat variants]

Descriptions:
D1: [Primary benefit + proof + CTA — 90 chars]
D2: [Secondary benefit + urgency — 90 chars]
D3: [Feature list + differentiator — 90 chars]
D4: [Social proof + CTA — 90 chars]
```

## Meta Ads (Facebook & Instagram)

### Specs (2026)
| Element | Limit | Best Practice |
|---|---|---|
| Primary text | 125 chars (preview), 500 max | Front-load the hook in first 125 chars |
| Headline | 27 chars (truncates on mobile) | Benefit or CTA, not brand name |
| Description | 27 chars | Secondary benefit or social proof |
| Image | 1080x1080 (1:1) or 1080x1920 (9:16) | Less than 20% text overlay |
| Video | 15-60 seconds | Hook in first 3 seconds |

### 3-Variant Framework
Write all three for every ad set:

**Variant A — Pain-focused (PAS)**
```
Primary: [State the pain]. [Agitate — make it vivid]. [Solution — your product + CTA].
Headline: [Outcome they want]
Description: [Proof point]
CTA: Learn More / Sign Up
```

**Variant B — Benefit-focused (BAB)**
```
Primary: [Before state]. [After state with your product]. [Bridge — how to get there].
Headline: [Key benefit]
Description: [Differentiator]
CTA: Get Started / Shop Now
```

**Variant C — Social proof / FOMO**
```
Primary: "[Customer quote or stat]". [Brief context]. [CTA].
Headline: [Join X others / Limited time]
Description: [Result metric]
CTA: Learn More / Sign Up
```

### Hook Formulas (first line of primary text)
- "Most [audience] don't realize [surprising fact]."
- "We analyzed [X data points]. Here's what we found."
- "Stop [common mistake]. Here's what to do instead."
- "[Stat]% of [audience] struggle with [problem]."
- "I [achieved result] in [time frame]. Here's how."

## LinkedIn Ads

### Specs (2026)
| Format | Intro Text | Headline | Body |
|---|---|---|---|
| Single Image | 150 chars (desktop truncation) | 70 chars | N/A |
| Carousel | 150 chars | 45 chars per card | N/A |
| Video | 150 chars | 70 chars | N/A |
| Message Ad | N/A | 60 chars subject | 500 chars body |
| Conversation Ad | N/A | N/A | 500 chars per message |

### LinkedIn Targeting Best Practices
- **Most effective**: Job title + company size + industry (AND targeting)
- **Audience size**: 50,000-500,000 for sponsored content
- **Exclude**: Competitors, current customers (upload exclusion list)
- **Lookalike**: Upload customer list for matched audiences

### LinkedIn Ad Copy Rules
1. Lead with insight or data, not a sales pitch
2. Speak to the professional identity ("As a [role], you know...")
3. Use first person for thought-leadership sponsored posts
4. Keep intro text under 150 chars — mobile truncates aggressively
5. CTA should match the funnel stage (awareness = "Learn More," conversion = "Request Demo")

### LinkedIn Message Ad Template
```
Subject: [Question about their pain point — 60 chars]

Hi {FirstName},

[1 sentence — reference a common challenge for their role].

[1 sentence — what you help with + proof point].

[1 sentence — the ask (single CTA)].

[CTA Button: "See How It Works" / "Book a Call"]

— [Your Name], [Title]
```

## General Ad Copy Principles

### The 3-Second Rule
On any platform, you have ~3 seconds to earn attention. The first line, headline, or opening frame must:
1. Be relevant to the audience (they see themselves in it)
2. Create curiosity or tension (they want to know more)
3. Promise a payoff (they believe it's worth their time)

### Copy Length by Funnel Stage
| Stage | Copy Length | Reason |
|---|---|---|
| Cold (awareness) | Short — hook + benefit + CTA | They don't know you; earn attention fast |
| Warm (consideration) | Medium — story + proof + CTA | They're evaluating; give them reasons |
| Hot (retargeting) | Short — reminder + urgency + CTA | They already know; just nudge |

### Power Words for CTAs
| Intent | Words |
|---|---|
| Urgency | Now, Today, Limited, Last chance, Before [date] |
| Value | Free, Bonus, Exclusive, Instant, Guaranteed |
| Ease | Simple, Easy, Quick, No-hassle, Done-for-you |
| Proof | Proven, Trusted, #1, Award-winning, As seen in |

### Numbers That Convert
- Odd numbers outperform even in headlines (7, 13, 27)
- Specific numbers build credibility ("47%" not "nearly 50%")
- Dollar amounts and time frames create tangible expectations
- "3 steps" or "5 minutes" reduce perceived effort
